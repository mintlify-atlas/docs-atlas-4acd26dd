Chronos Calendar includes a powerful todo management system that lives alongside your calendar, helping you manage tasks, create custom categories, and stay organized.

## Todo Sidebar Overview

The todo sidebar appears on the left side of the Chronos Calendar interface:

- **Category tabs** - Horizontal tabs at the top for switching between lists
- **Task input** - Quick-add field for creating new tasks
- **Task list** - Your todos organized by the selected category
- **User profile** - Account menu at the bottom

<Tip>
  The todo system is local to Chronos Calendar and does not sync to Google Tasks or Google Calendar. Your todos are stored securely in your Chronos account.
</Tip>

## Creating Tasks

### Quick Add Task

The fastest way to create a task:

<Steps>
  <Step title="Focus the Input Field">
    Click the task input field at the top of the todo sidebar, or press the `N` key from anywhere in the app.
  </Step>

  <Step title="Type Your Task">
    Enter a descriptive task title (e.g., "Review quarterly reports").
  </Step>

  <Step title="Press Enter">
    Press Enter to create the task. It's added to the current category immediately.
  </Step>
</Steps>

<Note>
  The `N` keyboard shortcut works globally throughout Chronos Calendar, except when you're already typing in an input field or textarea.
</Note>

### Adding Tasks to Specific Categories

When viewing the "All" category:
- Click the **+** button next to any category group header
- An input field appears within that category
- Type your task and press Enter
- The task is added to that specific category

When viewing a specific category:
- Tasks you create are automatically added to that category
- The category icon and name appear in the task input area

## Managing Tasks

### Completing Tasks

Mark tasks as done:

<Steps>
  <Step title="Click the Checkbox">
    Click the circular checkbox to the left of any task.
  </Step>

  <Step title="Completion Animation">
    The checkbox animates with a green checkmark, providing visual feedback.
  </Step>

  <Step title="Task Moves to Completed">
    Completed tasks automatically move to the "Completed" category and display with a strikethrough.
  </Step>
</Steps>

<Info>
  Tasks in the Completed category show a checkmark and grayed-out text. Clicking the checkbox on a completed task deletes it permanently.
</Info>

### Uncompleting Tasks

To mark a completed task as incomplete:
1. Navigate to the "Completed" category
2. Click the checkbox on any completed task
3. The task is deleted from Completed

<Warning>
  Clicking the checkbox on a task in the Completed category permanently deletes it. There is no undo. If you want to un-complete a task, you'll need to recreate it.
</Warning>

### Reordering Tasks

Organize tasks by priority using drag and drop:

<Steps>
  <Step title="Find the Drag Handle">
    Look for the "⋮⋮" handle on the right side of each task.
  </Step>

  <Step title="Drag the Task">
    Click and hold the drag handle, then drag the task up or down within the list.
  </Step>

  <Step title="Drop in Position">
    Release the mouse button to drop the task in its new position. The order is saved automatically.
  </Step>
</Steps>

<Note>
  Task order is maintained separately for each category. Reordering tasks in one category doesn't affect their order in other categories.
</Note>

## Categories and Lists

Organize your todos using categories (also called lists). Chronos Calendar includes default categories and lets you create custom ones.

### Default Categories

Chronos Calendar includes three built-in categories:

<CardGroup cols={3}>
  <Card title="Inbox" icon="inbox">
    Default category for new tasks when no specific category is selected
  </Card>
  
  <Card title="Today" icon="calendar-day">
    Tasks you want to focus on today
  </Card>
  
  <Card title="Completed" icon="circle-check">
    Automatically stores completed tasks
  </Card>
</CardGroup>

<Note>
  Default categories (Inbox, Today, Completed) cannot be deleted or renamed. They're protected system categories.
</Note>

### Creating Custom Categories

Create your own categories to organize tasks by project, context, or any system that works for you:

<Steps>
  <Step title="Click the Plus Button">
    In the category tabs at the top of the sidebar, click the **+** button on the right side.
  </Step>

  <Step title="Choose a Color">
    A color picker appears as a small circle. Click it to select from 8 available colors:
    - Blue
    - Purple  
    - Pink
    - Amber
    - Green
    - Teal
    - Orange
    - Red
  </Step>

  <Step title="Name Your Category">
    Type a name for your category (e.g., "Work Projects", "Personal", "Shopping").
  </Step>

  <Step title="Save the Category">
    Click the green checkmark button or press Enter to create the category.
  </Step>
</Steps>

<Tip>
  Category colors help you visually distinguish between different areas of your life or types of tasks. Choose colors that are meaningful to you.
</Tip>

### Renaming Categories

In the current version, custom categories cannot be renamed after creation. To change a category name:
1. Create a new category with the desired name
2. Move tasks to the new category (manually recreate them)
3. Delete the old category

### Deleting Categories

Remove categories you no longer need:

<Steps>
  <Step title="Right-Click the Category Tab">
    Right-click (or long-press on mobile) on any custom category tab.
  </Step>

  <Step title="Select Delete">
    A context menu appears with a "Delete" option marked in red. Click it.
  </Step>

  <Step title="Confirm Deletion">
    The category and all its tasks are permanently deleted.
  </Step>
</Steps>

<Warning>
  Deleting a category permanently deletes all tasks within it. This action cannot be undone. Consider moving important tasks to another category before deleting.
</Warning>

### Reordering Categories

Customize the order of your category tabs:

1. Click and hold any custom category tab (not "All")
2. Drag it left or right to reorder
3. Release to drop in the new position
4. The order is saved automatically

<Note>
  The "All" category cannot be reordered—it always appears first. Default system categories (Inbox, Today, Completed) can be reordered along with your custom categories.
</Note>

## Viewing Modes

### All View

The "All" category provides an overview of all your tasks:

- Tasks are grouped by their category
- Each category group can be expanded or collapsed
- Click the category header to toggle expansion
- See task counts for each category at a glance
- The collapse indicator (▼) rotates to show state

<Tip>
  Use "All" view for a high-level overview of everything on your plate. Great for daily planning and prioritization.
</Tip>

### Category View

When you select a specific category:

- Only tasks in that category are displayed
- The category name and color appear prominently
- Task count shows the number of tasks in the category
- Quick-add field creates tasks directly in this category
- Drag and drop reordering works within the category

### Completed View

The Completed category is special:

- Shows all tasks you've marked as complete
- Tasks display with checkmarks and strikethrough text
- Clicking a checkbox deletes the task permanently
- Collapsed by default in "All" view to reduce clutter
- Great for reviewing what you've accomplished

## Category Groups in All View

When viewing "All", tasks are organized into expandable groups:

### Group Headers

Each category group header shows:
- **Category icon** - The colored circle representing the category
- **Category name** - The name of the list
- **Task count** - Number of tasks in this category
- **Collapse indicator** - Arrow showing expanded/collapsed state
- **Add button** - Plus icon to quickly add tasks to this category

### Expanding and Collapsing

<Steps>
  <Step title="Click the Header">
    Click anywhere on a category group header to toggle it.
  </Step>

  <Step title="View State Change">
    - **Expanded** - Arrow points down (▼), tasks are visible
    - **Collapsed** - Arrow points right (▶), tasks are hidden
  </Step>

  <Step title="State Persists">
    Your expand/collapse preferences are remembered within your session.
  </Step>
</Steps>

<Note>
  The Completed category is collapsed by default to keep your todo sidebar focused on active tasks.
</Note>

## Task Organization Strategies

### By Project

Create categories for each project:
- "Website Redesign"
- "Marketing Campaign"
- "Home Renovation"

Benefits: Clear separation, easy to focus on one project

### By Context

Organize by where or how you do work:
- "Computer"
- "Phone Calls"
- "Errands"
- "At Home"

Benefits: Batch similar tasks together, work efficiently by context

### By Time

Use temporal categories:
- "Today"
- "This Week"
- "Someday"

Benefits: Natural prioritization, clear focus on what's urgent

### Hybrid Approach

Combine multiple strategies:
- Use Today for urgent items
- Use project categories for specific work
- Use Inbox as a catch-all

Benefits: Flexibility, captures different types of tasks

## Drag and Drop Details

### Within a Category

- Drag tasks up or down to change priority
- The task being dragged becomes slightly transparent
- Other tasks shift to make room
- Drop the task in the desired position
- Order is saved immediately

### Visual Feedback

During dragging:
- **Active task** - Reduced opacity (40%)
- **Drop zone** - Other tasks shift to show where the task will land
- **Cursor** - Changes to grab (grabbing) cursor
- **Shadow** - Task gains elevation with shadow when lifted

<Note>
  Tasks can only be reordered within their current category. Cross-category drag and drop is not currently supported.
</Note>

## Troubleshooting

### Tasks Not Saving

1. Check your internet connection
2. Try refreshing the page
3. Check the browser console for errors
4. Ensure you're still signed in (check the user profile icon)

### Tasks Disappearing

1. Check if you're viewing the correct category
2. Look in the Completed category—you may have accidentally checked it
3. Verify the category wasn't deleted
4. Check the "All" view to see if the task moved to another category

### Drag and Drop Not Working

1. Make sure you're clicking the drag handle (⋮⋮), not the task itself
2. Try clicking and holding for a moment before dragging
3. Check if you're viewing a single category (drag and drop doesn't work across categories in All view)
4. Ensure you're not trying to drag system category tabs (like "All")

### Category Colors Not Appearing

- Category colors should appear as dots next to category names
- If colors aren't showing, try refreshing the page
- Colors are stored with each category and should persist

## Advanced Features

### Task Order Persistence

Task ordering is stored per category:
- Each task has an `order` property
- Lower numbers appear first
- Order is recalculated when you drag and drop
- Order persists across sessions and devices

### Completed Category Behavior

The Completed category has special rules:
- Tasks moved here are marked as `completed: true`
- Clicking the checkbox deletes instead of un-completing
- New tasks cannot be created directly in Completed
- The category cannot be deleted or renamed
- It's collapsed by default in All view

### Account Integration

Your todos are tied to your Chronos Calendar account:
- Tasks sync across devices (web, desktop)
- Tasks are stored securely in the Chronos database
- Tasks are NOT synced to Google Calendar or Google Tasks
- When you sign out, tasks are no longer visible (but not deleted)
